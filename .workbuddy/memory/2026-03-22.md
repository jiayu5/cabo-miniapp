## 核心计分规则实现：
  1. 喊CABO赢了 → 本轮得0分
  2. 喊CABO输了 → 本轮手牌分×2
  3. 总分恰好=100 → 变为50分（每人只能触发一次）
  4. 有玩家总分>100 → 游戏结束，总分最低者获胜
